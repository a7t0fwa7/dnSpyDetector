# dnSpyDetector
This is a different but quick way to check for the presence of dnSpy hooks in memory.

![image](https://i.imgur.com/lj8K2j5.png)

## Supports:
- IsDebuggerPresent
- CheckRemoteDebuggerPresent
- System.Diagnostics.Debugger
- NtRaiseHardError
- CloseHandle
